## Getting Started

### Your First Email

1. **Open the Email Builder**: When you first open the builder, you'll see an empty canvas ready for your content.

2. **Choose a Template** (Optional): Click the **"Templates"** button in the top toolbar to browse predefined templates. You can:
   - Select a template to start with
   - Create a blank template
   - Save your current design as a template

3. **Start Building**: Drag blocks from the left sidebar onto the canvas to begin building your email.

---

## Understanding the Interface

The Email Builder has three main areas:

### 1. Left Sidebar (Blocks Palette)
This is where you'll find all available blocks you can add to your email. Simply drag a block from here onto the canvas.

**Available Blocks:**
- **Text**: Add formatted text content
- **Image**: Insert images
- **Button**: Create call-to-action buttons
- **Columns**: Create multi-column layouts
- **Divider**: Add horizontal lines
- **Spacer**: Add empty space
- **Social Links**: Add social media icons
- **Footer**: Add footer with company info
- **Header**: Add header with logo and menu

### 2. Canvas (Center Area)
This is your workspace where you build your email. You can:
- Drag blocks here to add them
- Click blocks to select and edit them
- Drag blocks up or down to reorder them
- Drag blocks outside the canvas to delete them

### 3. Right Sidebar (Properties Panel)
When you select a block, this panel shows all the properties you can edit for that block. Each block type has different properties you can customize.

## Working with Blocks

### Adding Blocks

1. **Find the block** you want in the left sidebar
2. **Click and drag** it onto the canvas
3. **Drop it** where you want it to appear

**Tip**: You can drop blocks between existing blocks to insert them in a specific position.

### Selecting Blocks

- **Click any block** on the canvas to select it
- Selected blocks show a **blue border** and a label at the top
- The right sidebar will show the properties for the selected block

### Moving Blocks

- **Click and drag** a selected block up or down to reorder it
- The block will move to the new position when you release it

### Deleting Blocks

**Method 1**: Drag the block outside the canvas area
**Method 2**: 
1. Select the block
2. Click the **trash icon** (üóëÔ∏è) in the right sidebar
3. Confirm deletion

### Editing Blocks

1. **Select the block** you want to edit
2. **Use the right sidebar** to change properties
3. Changes appear **instantly** on the canvas

## Using Templates

   - **Transactional Email**: Perfect for order confirmations and invoices
   - **Welcome**: Ideal for welcoming new users
   - **Newsletter**: Complete design for newsletters

1. Build your email design

## Editing Block Properties

Each block type has different properties you can customize. Here's what you can edit for each:

### Text Block

- **Content**: Use the rich text editor to format your text
  - **Bold**, *Italic*, <u>Underline</u>, ~~Strikethrough~~
  - Headings (H1, H2, H3)
  - Bullet and numbered lists
  - Text alignment (left, center, right)
  - Text color
  - Links
- **Alignment**: Left, Center, or Right
- **Font Size**: Size of the text
- **Color**: Text color
- **Padding**: Space around the text

### Image Block

- **Image URL**: Paste the URL of your image
- **Alt Text**: Description for accessibility
- **Width**: Image width (e.g., "100%", "300px")
- **Alignment**: Left, Center, or Right
- **Padding**: Space around the image

**Note**: If no image URL is provided, a placeholder showing "Image" will appear.

### Button Block

- **Button Text**: The text displayed on the button
- **URL**: The link the button goes to
- **Background Color**: Button background color
- **Text Color**: Color of the button text
- **Width**: Auto or Full Width
- **Alignment**: Left, Center, or Right
- **Border Radius**: Roundness of corners
- **Padding**: Space inside the button

### Columns Block

- **Column Count**: 2, 3, or 4 columns
- **Gap**: Space between columns
- **Padding**: Space around the columns
- **Background Color**: Background color of the columns area

**To add content to columns**: Drag blocks onto the columns block. They will automatically distribute across columns.

### Divider Block

- **Color**: Color of the divider line
- **Height**: Thickness of the line (e.g., "1px", "2px")
- **Padding**: Space above and below the divider

### Spacer Block

- **Height**: Amount of empty space (e.g., "20px", "50px")

### Social Links Block

- **Add Links**: Click "Add Link" to add social media icons
- **For each link**:
  - **Platform**: Choose from Facebook, X (Twitter), Instagram, LinkedIn, YouTube, or Custom
  - **URL**: Your social media profile URL
  - **Custom Icon URL**: (Only for Custom platform) URL to your custom icon
- **Icon Size**: Size of the icons (e.g., "24px", "32px")
- **Icon Color**: Color of the icons
- **Spacing**: Space between icons
- **Alignment**: Left, Center, or Right
- **Padding**: Space around the social links

### Footer Block

- **Company Name**: Your company name
- **Company Address**: Your company address
- **Copyright Text**: Copyright notice (e.g., "¬© 2025 Your Company")
- **Links**:
  - **Unsubscribe URL**: Link for users to unsubscribe
  - **Privacy Policy URL**: Link to privacy policy
  - **Terms URL**: Link to terms of service
  - **Show/Hide**: Toggle switches to show or hide each link
- **Styling**:
  - **Background Color**: Footer background color
  - **Text Color**: Color of footer text
  - **Font Size**: Size of footer text
  - **Padding**: Space around footer content

### Header Block

- **Logo**:
  - **Logo URL**: URL of your logo image
  - **Logo Alt Text**: Description for accessibility
  - **Logo Width**: Width of logo (e.g., "150px", "200px")
  - **Logo Height**: Height of logo (e.g., "auto", "60px")
- **Navigation Menu**:
  - **Show Menu**: Toggle to show/hide navigation menu
  - **Menu Items**: Click "Add Item" to add navigation links
    - **Label**: Text displayed in menu (e.g., "Home", "Products")
    - **URL**: Link destination
- **Styling**:
  - **Background Color**: Header background color
  - **Alignment**: Left, Center, or Right
  - **Padding**: Space around header content

**Note**: If no logo URL is provided, a placeholder showing "Logo" will appear.

## Tips & Best Practices

### Design Tips

1. **Start with a Template**: Use predefined templates as starting points - they're professionally designed and tested.

2. **Use Headers and Footers**: Always include a header with your logo and a footer with unsubscribe links for professional emails.

3. **Add Social Links**: Include social media icons in your emails to grow your following.

4. **Keep it Simple**: Don't overcrowd your email. Use spacers to create breathing room.

5. **Mobile-Friendly**: Test your email on mobile devices. The builder creates responsive designs, but always preview before sending.

6. **Use Columns Wisely**: Columns are great for newsletters, but keep them to 2-3 columns maximum for readability.

7. **Button Best Practices**:
   - Use clear, action-oriented text (e.g., "Shop Now", "Learn More")
   - Make buttons prominent with contrasting colors
   - One primary button per email is usually best

8. **Image Guidelines**:
   - Use high-quality images
   - Optimize images for web (keep file sizes reasonable)
   - Always include alt text for accessibility
   - Use absolute URLs (full web addresses, not relative paths)

### Content Tips

1. **Clear Subject Lines**: While not part of the builder, remember to write compelling subject lines.

2. **Personalization**: Use merge tags from your email service provider for personalization.

3. **Call-to-Action**: Make your primary action button prominent and easy to find.

4. **Unsubscribe Link**: Always include an unsubscribe link in your footer (required by law in many regions).

## Troubleshooting

### Block Not Appearing

**Problem**: You dragged a block but it's not showing on the canvas.

**Solution**: 
- Make sure you're dropping it on the canvas area (center section)
- Try refreshing the page
- Check that the block was actually added (it might be below the visible area)

### Can't Edit Block Properties

**Problem**: The right sidebar is empty or not showing properties.

**Solution**:
- Make sure you've clicked on a block to select it
- Look for the blue border around the selected block
- Try clicking the block again

### Image Not Showing

**Problem**: Image shows as broken or placeholder.

**Solution**:
- Check that the image URL is correct and complete (must start with http:// or https://)
- Ensure the image is publicly accessible (not behind a login)
- Try opening the image URL in a new browser tab to verify it works
- Use absolute URLs (full web addresses), not relative paths

### Can't Delete a Block

**Problem**: Block won't delete when dragged outside.

**Solution**:
- Make sure you're dragging it completely outside the canvas area
- Try using the delete button in the right sidebar instead
- Select the block first, then click the trash icon

### Changes Not Saving

**Problem**: Your edits disappear when you refresh.

**Solution**:
- This depends on how the application is configured
- Check with your developer/administrator about saving functionality
- Consider exporting your template as JSON to save it manually

### Template Not Loading

**Problem**: Selected template doesn't appear.

**Solution**:
- Wait a moment for the template to load
- Try selecting the template again
- Refresh the page and try again

### Social Links Not Working

**Problem**: Social media icons don't link correctly.

**Solution**:
- Check that you've entered complete URLs (starting with https://)
- Verify the URLs are correct by testing them in a browser
- Make sure you've selected the correct platform for each link

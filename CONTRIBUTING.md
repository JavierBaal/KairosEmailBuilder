# Contributing to Kairos Email Builder

Thank you for your interest in contributing to Kairos Email Builder! We welcome contributions from the community to help make this the best lightweight email builder for React.

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/YOUR_USERNAME/kairos-email-builder.git
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Create a branch** for your feature or bugfix:
    ```bash
    git checkout -b feature/amazing-feature
    ```

## Development Guidelines

*   **Language:** All code, comments, and documentation must be in **English**.
*   **Code Style:** We use ESLint and Prettier. Please ensure your code passes linting before submitting.
*   **Commits:** Use [Conventional Commits](https://www.conventionalcommits.org/) for your commit messages.
    *   `feat: add new button block`
    *   `fix: resolve drag issue on mobile`
    *   `docs: update readme`

## Pull Request Process

1.  Ensure your code builds and runs locally.
2.  Update the `README.md` if you are adding new features or changing usage instructions.
3.  Submit a Pull Request to the `main` branch.
4.  Provide a clear description of your changes and link to any relevant issues.

## Code of Conduct

Please note that this project is released with a [Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
